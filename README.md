# kover.js

Use JavaScript to implement CSS `background-size: cover`

## Usage

```html
<script src="path/to/jquery"></script>
<script src="path/to/kover.js"></script>
<script>
  $(function () {
    $.kover({
      wrapCell: '.test-block'
    })
  })
</script>
```
